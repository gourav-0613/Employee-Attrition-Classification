# Employee Attrition Classification System

## Overview
This repository contains a predictive Machine Learning pipeline designed to classify and identify high-risk factors leading to employee turnover. By analyzing HR metrics such as job satisfaction, salary, work-life balance, and performance ratings, this system provides actionable insights for HR departments to preemptively address employee attrition.

*Note: This project is currently under active development. Final model evaluation and deployment scripts are being pushed shortly.*

## Tech Stack
* **Language:** Python 3.x
* **Data Processing & EDA:** Pandas, NumPy
* **Machine Learning:** Scikit-learn (Decision Trees, Classification metrics)
* **Visualization:** Matplotlib, Seaborn

## Methodology (In Progress)
1. **Data Wrangling:** Handling missing values, encoding categorical HR variables, and scaling numerical features.
2. **Exploratory Data Analysis (EDA):** Identifying correlations between employee tenure, income, and attrition rates.
3. **Model Training:** Implementing classification algorithms (Decision Trees) to predict the binary outcome (Attrition: Yes/No).
4. **Evaluation:** Optimizing for precision and recall to mitigate the impact of class imbalance in HR datasets.

## Setup & Installation
To run this project locally once the source code is fully published:
1. Clone this repository.
2. Install the required dependencies: `pip install -r requirements.txt`
3. Execute the Jupyter Notebook `analysis.ipynb` to view the data pipeline and model training process.
